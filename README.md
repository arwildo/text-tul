# TexTul
<p align=center>

</p>

React App for text manipulation. 
* Convert Text to uppercase.
* Convert text to lowercase.
* Capitalize all the first letters on every word.
* Reverse the text from back to front.

## Built With

* React 16
* Webpack 5.1.3
* Tailwind 1.9.4
